1. 在DW菜单栏点击【插入】，可以新建各种html标签。

2. html标签一般是成对存在的，藏在< >和</ >中间的是它的显示内容。标签之间可以互相嵌套。 标签头< >内可以定义属性，不同的属性用空格隔开。
  例如，在a标签【<a href="z-page001.html">第一篇文</a>】中，href是这个标签的属性名，“=”后的是属性值；【第一篇文】是这个标签的显示内容。

3. 如果打算在html网页上写文章的话，先写好，把转义字符替换后，再粘贴到DW里面。

4. 经常见到的代码。

&quot; "
&amp; &
&lt; <
&gt; >
&ensp; 半角空格
&emsp; 全角空格
<br> 换行符
<p>  </p> 新段落
<table border="1">  </table> 带框线的表格
<tr>  </tr> 表格的行分隔符
<td>  </td> 表格的列分隔符

9. 网址的路径那个斜杠，是和问号共用一个按键的【/ 】，叫做【斜杠】，它也可以表示除号。和汉语顿号共用按键的【\】叫做【反斜杠】，某些编程语言会用它来表示转义。

  1. 行内元素如<span>，可以和其他元素共用浏览器的一行显示。块状元素如<p>在浏览器中独占一行。

  2. 有序列表用<ol>标签表示，是order list的意思，他会将里面的各个元素用编号标起来。有序列表的元素用<li>表示。我第10项之后都用了ol标签了。
    代码是这样的：
    <ol start="10">  <li>文本1</li>  <li>文本2</li>  </ol>

  3. 无续列表用<ul>标签表示，元素也是<li>。但是列表元素之前的符号会变成圆点。
    代码如下：
     <ul >  <li>文本1</li>  <li>文本2</li>  </ul>

    4. 显示效果：
    • 文本1
    • 文本2

  4. <dl> 标签用于创建定义列表。它是由定义标题和定义描述两部分组成的，具体语法格式如下：

    <dl>
          <dt>定义标题<dt>
          <dd>定义描述<dd>
          <dd>定义描述<dd>
    </dl>

    显示效果如下：

    定义标题
    定义描述
    定义描述

  5. 绝对路径和相对路径。【本地绝对路径】从盘符开始，到文件名称结束；【网络绝对路径】从网站的域名开始，到文件名结束，在使用时需要加上协议。【相对路径】以当前文件位置为参考点，到文件名称结束，其中，【./】代表当前位置（可省略），【../】 表示上一级目录， 【../../ 】表示上两级目录，以此类推。

    绝对路径：
    ①D:/Hbulider/HBuilder/tools/nview/index.js
    ②C:/Users/admin/Desktop/C语言中文网/url/url.html
    ③http://c.biancheng.net/view/7410.html
    相对路径：
    ①./html/login.html
    ②html/login.html
    ③../html/login.html

  6. 定义锚点。在网页恰当的标签里面加上【id】这个属性。
    像这样子：
    <h2 id="index">这是h2标题</h2>

  7. 锚点链接，可以跳转到指定页面中指定位置的链接，用<a>标签表示。【#】后跟上定义好的锚点id，id的top代表顶部（省略也行）。

    跳到当前页面是这样的：
    <a href=<"#top">回到顶部</a> 
    或者
    <a href="#">回到顶部</a>

    跳到其他页面是这样的：
    <a href="./index.html#index">跳转到index页面</a>

  8. <form>表单标签，<form> 标签内部有 4 种元素，分别是 input、option、select 以及 textarea 元素。
    它有三个常用属性，分别是action（表单提交到的位置，值为“提交到的url”）、method（表单提交的方法，值为post或者get）、name（表单的名称，不能带特殊字符和空格，不会显示在屏幕上）。
    get的提交方法：表单呈现到网页的地址栏上，不安全。
    post的提交方法：相对安全，数据会直接传到后台。

  9. <input>标签。它是自闭合标签，就是说它的内容一个<>能写完，没有</>这个尾巴。它的type属性值有很多种，包括：
    text（单行文本框）、password（密码框）、radio（单选按钮）、checkbox（复选框）、hidden（隐藏域）、file（文件上传域）、button（普通按钮）、submit（提交按钮）以及reset（重置按钮）等。
    下面有个代码示范：
    <body>
        <form action="http://z-page001.html" method="get" name="myForm">
            用户名:<input type="text" name="name"><br/>
            <input type="button" value="确定">
            <input type="reset" value="重置">         
        </form>
    </body>

    用户名:

  10. 单行文本框和密码框都有maxlength（最多允许输入的字符数）、size（最多能显示的字符数）、value（默认值，用户可输入更改）这三个属性。
    不同之处在于，密码框输入的文本会被转换成圆点显示。

  11. 单选按钮标签<input type="radio" />：
    当它有checked属性时，即表示这个选项被选中了。
    当一组的单选按钮标签的name属性是不相同时，便变成不正常的单选按钮（可以多选且选中不可取消）。
    它的value属性，用于在表单中判断用户选中了哪一项。

  12. 复选框：当一组复选项的name属性相同，其中一项选中了的话剩下的也会自动被选中。

  13. 文件上传域：只能用post方式上传文件。
    它的enctype属性必须设置为multipart/form-data文件才能正常提交。
    使用 accept 属性可以规定上传文件的类型。如【accept="image/png"】
    若设置了multiple属性，可以一次性上传多个文件。

  14. 常用的文件类型

     .jpg  image/jpg
     .png  image/png
     .gif  image/gif
     .jpeg  image/jpeg
     .html  text/html
     .css  text/css
     .js  text/javascript、application/javascript
     .txt  text/plain
     .zip  application/zip
     .mp4  audio/mp4、video/mp4

  15. 多行文本框<textarea>：又叫做文本域。当涉及到描述信息，内容比较多时，单行文本框很有可能放不下所有的内容，这时就需要用到多行文本框。
    文本域的cols属性：指文本域单行显示的字符数，又称文本域的可见宽度。
    文本域的rows属性：指文本域可现实的行数，又称文本域的可见长度。
    我准备在本网页下部的加个文本域做评论区。不过没有后台程序，所以本文的按钮都是摆设。

    具体语法格式如下：
    <select>

  17. 按钮：value属性值是按钮的显示名称。提交按钮的默认值是“提交”，重置按钮的默认值是“重置”。

  18. 图像按钮：<input type="image" src="url" >这样表示。显示出指定图像的按钮。它的属性有：
    1.  alt属性用来对图像进行文本说明。一般情况下，当浏览器因某种原因无法载入图像时，就会显示这段文本。
    2.  width 属性表示图片的宽度，默认单位是 px，不需要显示指明。
    3.  height 属性表示图片的高度，同 width 属性一样，默认单位也是 px，不需要显示指明。
